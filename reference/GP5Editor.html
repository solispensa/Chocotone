Help

This is a web editor for the Valeton GP5. You have to use Chrome or another browser with Web MIDI enabled, and a usb data cable.

You can use the keyboard to control some functions:

  • On/off effects: 0 to 9
  • Tap tempo: space
  • Previous and next patches: pgUp, pgDown

You can save the changes you make on the patch but it's only to the same patch you working on it. This is to prevent problems with the keybindings of tap tempo and effects.

Some features are not available in the USB , as they are not notified by the GP-5, such as real time changes to Global parameters, foot mode and CTL changes.
