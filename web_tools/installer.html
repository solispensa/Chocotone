üîß Troubleshooting
ESP32 not showing in list

Use a data cable, not charge-only. Try another USB port. Windows users: install CP210x drivers.

Installation fails / Packet Error

Hold the BOOT button on your ESP32 while clicking "Install". Release after "Erasing" appears.

"Failed to connect"

Close Arduino IDE or other serial monitors. Unplug/replug the ESP32 and try again.
